    • What is HTML and CSS?

    1. What is HTML?
      • HTML stands for Hypertext Markup LanguageC
      • HTML is used to create Web documents to be used online, every page you see on the internet is made from HTML
      • HTML was developed by Tim Berners-Lee at CERN using SGML in 1992 and from that point on, html has evolved from using a few tags to having a complex system of markup
      • HTML is written through a bunch of tags with each tag telling the browser what to do with the text on the webpage.
    2. What is CSS?
      • CSS stands for Cascading Style Sheets
      • CSS is used to describe what a webpage looks like
